# Elytra Swap Mod

Мод для Minecraft 1.16.5 Fabric, который позволяет быстро менять элитры и щиты между слотом нагрудника и offhand.

## Функции

- **Свап Элитр**: Быстро поменяйте элитры между слотом нагрудника и offhand
- **Свап Щита**: Быстро поменяйте щит между offhand и слотом нагрудника
- **Настраиваемая клавиша**: Привяжите свою любимую клавишу в настройках управления

## Установка

1. Скачайте Fabric Loader для версии 1.16.5
2. Поместите `elytra_swap-1.0.0.jar` в папку `mods`
3. Запустите игру

## Использование

1. Нажмите привязанную клавишу (по умолчанию `G`)
2. Выберите желаемую операцию в меню:
   - **Swap Elytra** - поменять элитры
   - **Swap Shield** - поменять щит
   - **Cancel** - закрыть меню

## Конфигурация

Вы можете переназначить клавишу в `Options > Controls > Elytra Swap`

## Зависимости

- Minecraft 1.16.5
- Fabric Loader 0.12.12+
- Fabric API 0.33.0+
